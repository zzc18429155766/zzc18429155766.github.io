🛠️ Free Open Source Tools

git-ai-commit

Generate beautiful, conventional commit messages from your staged diffs using AI. Supports OpenAI, Claude, and local models.

pip install git-ai-commit

ai-code-reviewer

AI-powered code review in your terminal. Get instant feedback on code quality, security issues, and best practices.

pip install ai-code-reviewer

git-smart

Smart git commands enhanced with AI. One command to stage, commit, and push with the perfect message.

pip install git-smart

ai-readme

Auto-generate professional README files for your projects. Analyzes your code and creates comprehensive documentation.

pip install ai-readme

ai-changelog

Automatically generate changelogs from git history using AI. Keep your users informed with beautiful release notes.

pip install ai-changelog

code-stats

Code statistics and analysis. Understand your codebase with detailed metrics, language breakdowns, and complexity scores.

pip install code-stats

git-timeline

Visual git history timeline. See your project's evolution with beautiful visualizations.

pip install git-timeline
